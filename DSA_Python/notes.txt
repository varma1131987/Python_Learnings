What is DSA?

DSA = Data Structures + Algorithms.

•	Data Structures: Ways to organize data (arrays, lists, trees, graphs, hashes).
•	Algorithms: Step-by-step procedures to process data (searching, sorting, traversal).

Why its required?

Because the same problem solved inefficiently might take hours; solved efficiently, microseconds. Big O notation measures this efficiency."


Big O Notation: The Cheat Sheet
--------------------------------

Notation	 Name	        Example                     	Speed
O(1)	    Constant	    Access array element by index	Fastest
O(log n)	Logarithmic	    Binary search	
O(n)	    Linear  	    Find max in unsorted array	
O(n log n)	Linearithmic	Merge sort, quick sort	
O(n²)	    Quadratic	    Bubble sort, nested loop	
O(n³)	    Cubic       	Triple nested loop	
O(2^n)	    Exponential	    Brute force recursion	
O(n!)	    Factorial	    Generate all permutations   	Slowest
